# BCT C++ Implementation

This directory contains the C++ source code for the Bayesian Context Trees (BCT) algorithm, adapted from the original repository by Ioannis Papageorgiou.

**Source:** [https://github.com/IoannisPapageorgiou/Bayesian-Context-Trees](https://github.com/IoannisPapageorgiou/Bayesian-Context-Trees)
**R Package:** [https://cran.r-project.org/package=BCT](https://cran.r-project.org/package=BCT)
